# Orientação a Objetos

## Cadastro de Pessoas

>FORM
<p>Implementação de uma aplicação que faça cadastro de pessoas.</p>
<p>A aplicação foi escrita em Python, utilizando o Flask Framework, Vue.js e MongoDB.</p>

### Linux Dependencies

> Python 3.6

> Flask 1.1.1

> MongoDB 3.6.8

## Build

#### Python

```bash
	$ sudo pip3 install pipenv
	$ pipenv install
	$ pipenv shell
	$ python3 app.py
```
### Frontend

```bash
	$ npm install
	$ npm run dev
```
